# behance-builders (ADC Hackathon 2025)

## 🚀 Overview
Split-The-Cost is a web application that helps users track and divide shared expenses easily (e.g., for roommates, trips, or group outings). The app enables users to log expenses, specify participants, and calculate who owes what in a user-friendly way.

## ✨ Features
- **User Signup/Login**: Secure authentication system.
- **Group Creation**: Users can create and manage groups for different expense categories.
- **Expense Logging**: Users can add expenses and specify participants.
- **Balance Tracking**: The app calculates and displays who owes what.

### Additional
- **Advanced Cost Splitting**: Option to split costs unevenly.
- **Payment Settlement**: Users can mark payments as settled.
- **Expense Categories**: Organize expenses with custom categories.

## 🛠️ Tech Stack
### Frontend
- **Framework**: React + TypeScript
- **State Management**: Redux Toolkit
- **Data Fetching**: React Query
- **Form Handling**: React Hook Form
- **Validation**: Zod
- **Styling**: Tailwind CSS

### Backend
- **Runtime**: Node.js
- **Database & Authentication**: Firebase
